The Function

Start with a simple question: what happens when you compute n(n+1) / (n+1)n for positive integers?

  • G(1) = 1/2
  • G(2) = 4/9 ≈ 0.444
  • G(3) = 27/64 ≈ 0.422
  • G(4) = 256/625 = 0.4096

Integers in, rationals out. No transcendentals. The function decreases monotonically toward 1/e ≈ 0.3679 as n grows. But the structure hidden in these simple rationals turned out to be far richer than expected.

The Constant

Define AG = Σ G(n)/n! for n = 1 to infinity. This sum converges to:

AG ≈ 1.244331783986725...

We computed this to 10 million digits using mpmath on a Raspberry Pi 5 with a Hailo-8 accelerator. The computation took about 4 hours. The full 10M-digit file is 9.5MB.

As of this writing, this constant does not appear in OEIS or any published constant database we've checked.

50+ Identities

What makes G(n) interesting isn't any single property — it's how many different mathematical structures it connects:

  • Telescoping product: Π G(k) from k=1 to n = (n!)2 / (n+1)n
  • Ratio formula: G(n)/G(n-1) = (n²/(n²-1))n
  • Negative extension: G(-n) = -G(n-1) — the function is antisymmetric
  • Logarithmic decomposition: ln G(n) = (n+1)ln(n) - n·ln(n+1)
  • Second differences: always negative (concavity preserved)
  • GCD property: gcd(n(n+1), (n+1)n) = 1 for all n (coprimality)
  • Fibonacci connection: G(Fk) approaches 1/e at the golden ratio rate

Every identity has been verified numerically across hundreds of values. The algebraic proofs are collected in the FRAMEWORK.md document.

The Crossover Point

There's a critical value α ≈ 2.293166 where the behavior of G(n) changes character. Below this point, nn grows faster than (n+1)n. Above it, the roles reverse. This crossover appears in the consecutive product formula and connects to the transition between discrete and continuous regimes.

Connections

G(n) touches several areas of mathematics:

  • Riemann: G(1) = 1/2, the critical line value
  • Quantum mechanics: The density matrix trace condition (tr(ρ) = 1) maps to the product formula
  • DNA base counting: The product ΠG(k) crosses 1 at exactly n = 4 (the number of DNA bases)
  • Entropy: W = n(n+1) + (n+1)n has Boltzmann-like structure
  • Cayley trees: G(n) counts weighted labeled trees on n+1 vertices
