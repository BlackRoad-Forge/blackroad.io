The Problem

I run BlackRoad OS — a one-person company with 18 domains, 273 repositories, 200 AI agents, and a 5-node Raspberry Pi cluster. I needed a way to search across all of it. Not just file names — I wanted full-text search with AI-powered summaries.

Every hosted search service I looked at was either too expensive, too limited, or required sending my data to someone else's servers. I wanted something that runs on my hardware, indexes my content, and answers questions about my own ecosystem.

The Architecture

The search engine is a single Cloudflare Worker backed by a D1 database (SQLite at the edge). Here's how it works:

  • Indexer: A cron job crawls all 18 domains, 273 GitHub repos, and 200 RoundTrip agents. Each page gets its title, description, full text, domain, category, and tags stored in D1.
  • FTS5: SQLite's full-text search extension handles queries. It's surprisingly fast — 500ms average including network round-trip from anywhere in the world.
  • AI Answers: For every query, the search engine also generates a one-paragraph AI summary using Ollama running on the Pi cluster. If Ollama is slow, it falls back to Cloudflare Workers AI.
  • Categories: Results are tagged into categories (repo, agent, site, docs, product, infra, app, tech) so you can filter.

The Hardware

The "5 Raspberry Pis" part is real, but the search engine itself runs as a Cloudflare Worker. The Pis handle everything else:

  • Alice (Pi 4, 8GB) — Gateway, Pi-hole DNS, PostgreSQL, Redis, Qdrant vector DB
  • Cecilia (Pi 5, 8GB + Hailo-8) — Primary Ollama inference, MinIO object storage
  • Octavia (Pi 5, 8GB + Hailo-8) — Gitea (239 repos), Docker, NATS messaging
  • Aria (Pi 4, 4GB) — Backup and monitoring
  • Lucidia (Pi 4, 8GB) — 334 web apps via nginx, PowerDNS, GitHub Actions runners

Plus two small DigitalOcean droplets for TLS termination (Caddy) and edge routing. Total cost: $38/month.

What's Indexed

The search engine currently indexes:

  • 1,764 GitHub repositories across 16 organizations
  • 200+ AI agents from the RoundTrip coordination system
  • 35 pages on blackroad.io
  • 18 domain homepages with full content
  • Docs, products, infrastructure pages

The AI Answer Box

The most interesting feature is the AI answer box. When you search for something, the engine doesn't just return links — it generates a brief, grounded answer based on the actual indexed content.

The AI pipeline: query → FTS5 results → top 5 snippets → prompt to Ollama (llama3.2:1b on Cecilia) → one-paragraph answer. If Ollama is cold or slow, it falls back to Workers AI (llama-3.1-8b). The answer always cites actual URLs from the index — no hallucination.

What I Learned

  • SQLite FTS5 is underrated. It handles 2,000+ documents with sub-second queries. No Elasticsearch needed.
  • D1 is perfect for this. Cloudflare's edge SQLite means the search index is replicated globally. Queries from Tokyo are as fast as queries from New York.
  • AI answers need grounding. The answer box only works because it's fed real snippets from real results. Without grounding, it would hallucinate.
  • $38/month is enough. You don't need AWS. Five Pis, two small VPS nodes, and Cloudflare's free tier gets you surprisingly far.
